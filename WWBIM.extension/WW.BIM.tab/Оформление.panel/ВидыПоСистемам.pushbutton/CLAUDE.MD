Ключевые решения
1. Один выпадающий список для категории
Вместо вкладок — простой ComboBox: Воздуховоды / Трубопроводы / Кабельные лотки / Все MEP-системы
2. Плоский список систем с чекбоксами
Без деревьев и вложенности — пользователь сразу видит все системы выбранной категории
3. Три кнопки быстрого выбора
Экономят клики при работе с большими списками
4. Минимум настроек
Только то, что реально нужно:

Шаблон вида (берётся из проекта)
Префикс для имени (чтобы виды группировались в браузере)
Две опции-чекбокса

5. Одна главная кнопка действия
«Создать» — сразу понятно, что произойдёт
Поведение

При смене категории список обновляется мгновенно
Имя вида формируется автоматически: {Префикс}{Имя системы}
Если вид с таким именем существует — добавляется номер
После создания — короткое уведомление внизу окна: «Создано 3 вида»
